# 天氣購物車

安裝：

```
$ git clone https://github.com/uio456/shopping_cart.git
$ cd shopping_cart
$ bundle install
```

建立種子資料

```
$ rails db:migrate
$ rails db:seed
$ rails dev:fake_products
```

# User stories

### 使用者功能

- 可根據「今日、明日天氣」推薦購買商品
- 可把商品加入收藏
- 可瀏覽自己所有收藏的商品
- 享有折扣方式
  - 特定商品滿 x 件折 z %
  - 特定供應商的場品滿 x 件折 z %
  - 訂單滿 x 元折 z %
  - 訂單滿 x 元免運費
  - 折扣可限定總共優惠 Y 元
  - 可限定日期後不再有任何折扣

### 後台管理介面 (區分為自家管理員、廠商管理員)

###### 自家管理員帳號：root@example.com 密碼：12345678
###### 廠商管理員帳號：eslite@123 密碼：六個1 (方便測試，無限制email格式)

- 新增、編輯廠商
  - 自家管理員可操作所有廠商
  - 廠商管理員無權限

- 新增、編輯商品
  - 自家管理員可操作所有商品
  - 廠商管理員可操作自家商品

- 新增、編輯管理員
  - ​自家管理員可操作所有管理員
  - 廠商管理員可操作自家管理員

- 調整優惠
  - 自家管理員可操作優惠
  - 廠商管理員無權限

- 編輯顧客
  - 自家管理員可操作所有顧客
  - 廠商管理員無權限

- 管理根據天氣推薦商品
  - 自家管理員可操作天氣推薦商品
  - 廠商管理員無權限
